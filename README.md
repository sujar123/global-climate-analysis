# Global Climate Change Analysis

Analysis of global temperature trends across 7 regions using statistical methods in Python.

## Research Questions
- Did temperatures increase significantly after 2010?
- Are there significant temperature differences across regions?
- Is there a positive long-term warming trend over time?

## Methods
- **T-tests** — compared average temperatures before and after 2010 per region
- **One-way ANOVA** — tested for significant temperature differences across regions
- **Pearson Correlation** — measured the relationship between year and temperature over time
- **Visualizations** — line charts and scatterplots with fitted trendlines per region

## Dataset
Global temperature dataset covering 7 regions: Africa, Asia, Australia/South Pacific, Europe, Middle East, North America, South/Central America & Caribbean (7,964 records after cleaning)

## Tools
Python, Pandas, Matplotlib, SciPy
